# 📝 Fullstack Notes App

Una aplicación fullstack para gestión de notas con autenticación, backend API y base de datos en PostgreSQL.

---

## 🚀 Tecnologías

### Frontend
- React (Vite)
- React Router
- Zustand
- Clerk Auth
- HTML, CSS

### Backend
- Arquitectura MVC
- Node.js
- Express
- Clerk Middleware
- CORS
- Webhooks
- zod
- Prisma

### Base de datos
- PostgreSQL (Vercel Storage)

### Deploy
- Vercel (Frontend + Backend)

---

## 📦 Características

- 🔐 Autenticación con Clerk
- 📝 Crear, editar y eliminar notas
- 📌 Sistema de tareas
- 🔄 API REST
- 🌐 Deploy fullstack en Vercel
